## How It Works

1. Provide creative prompt.
2. OpenAI enhances prompt with trending context.
3. Veo3 generates TikTok-optimized scenes.
4. FFmpeg merges scenes into a cohesive video.
5. Auto-publish to TikTok.

## Getting Started

```bash
# install deps
npm i

# run app
npm run dev
```
